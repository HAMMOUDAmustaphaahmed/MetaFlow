Actions CRUD par phase
C BE — Créer Plan, Projet, Note
U BE — Réviser plan, corriger
D BE — Annuler plan refusé
C Prod — Ouvrir OF, affecter
U Prod — Modifier étapes OF
D Prod — Supprimer tâche inutile
C Liv — Créer BL, réception
U Liv — Modifier BL, réserves
D Liv — Annuler BL si avortement
Relations UML
▷ Plan → Charpente, Cabine
▷ OF → FabCharpente, FabCabine
▷ Equip → Sanit., Elec., Mob.
▷ User → RespBE, ChefProd, etc.

◆ FabCabine ◆→ Menuiserie
◇ FabCabine ◇→ Equipement

→ SignalFaute → Ouvrier(s)
